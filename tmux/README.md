# Install TPM

```sh
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

# Directory:

```sh
cp .tmux.conf ~/
cp .config/tmuxinator/asdf.yml ~/.config/tmuxinator
```

# type this in terminal if tmux is already running

```sh
tmux source ~/.tmux.conf
```

# Install Plugins

- `prefix` = `C-Space`
- `prefix - r` to reload
- `prefix - I`
